# CGSS
Crystal Grid Solar Solutions

This repository contains the source code for Crystal Grid Solar Solutions' web presence and digital tools. We're dedicated to making solar energy accessible and affordable for homes and businesses.

## About Us

Crystal Grid Solar Solutions provides comprehensive solar installation services, from initial consultation and system design to installation and ongoing maintenance. Our mission is to accelerate the transition to clean, renewable energy.

## What's Inside

- Company website and portfolio
- Solar calculator tools
- Project documentation
- Client resources

## Getting Started

```bash
# Clone the repository
git clone https://github.com/jashjson/cgss.git

# Navigate to project directory
cd cgss

# Install dependencies
npm install

# Run development server
npm run dev
```
